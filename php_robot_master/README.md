# PHP\_Robot文档📋

* * *

它是一个基于`CQhttp`的QQ机器人框架，全部由PHP编写。

采用`SG11`加密核心文件，对CQ码提供了全部支持！

您可以使用它快速部署一个QQ机器人~

## 环境配置💻

* * *

*   PHP7.0 安装 SG11扩展
*   放行分发器和主控的端口
*   配置伪静态
*   配置主控器的`Config.yml`与分发器的`Config.json`

### **伪静态**

* * *

```
location ~ \.(json)$ {
    deny all;
}
location / {
  try_files $uri $uri/ $uri.php?$args;
}
```

## 故障排除❓

* * *

如果您服务器无法部署此项目？

1.  在go-cqhttp根目录下找到runoob.log，检查程序运行日志。
2.  检查端口放行情况（阿里腾讯等需要编辑安全组，不能在宝塔直接更改）

如果还是无法部署，请加入🐧群进行询问！
